# Папка для загрузки в GitHub

**Эта папка** — чистая копия проекта `1_дата-аналитика` без секретов и локальных файлов.

## Что здесь есть

- ✅ Весь код проекта (Python, SQL)
- ✅ GitHub Actions workflow (папка `.github` - стандартная для GitHub)
- ✅ Документация
- ✅ requirements.txt
- ✅ `.gitignore` (стандартное имя для Git)
- ❌ Без файла `.env` (секреты)
- ❌ Без локальных файлов (.DS_Store, __pycache__ и т.д.)

## Скрытые файлы

В этой папке остались только **необходимые** скрытые файлы (стандарт для Git/GitHub):
- `.github/` — стандартная папка для GitHub Actions workflow (GitHub требует именно это имя)
- `.gitignore` — стандартное имя файла, Git автоматически его использует

Все остальные скрытые файлы (`.env`, `.DS_Store`, `.git` и т.д.) исключены.

## Как использовать

1. **Первый раз:** Загрузи эту папку в GitHub репозиторий
2. **При изменениях:** Обновляй эту папку из основной папки `1_дата-аналитика`
3. **Коммить:** Делай коммиты и пуш в GitHub из этой папки

## Обновление папки

Когда в основной папке `1_дата-аналитика` появляются изменения, обнови эту папку:

```bash
cd /Users/nikitasmirnov/Desktop/cursor/Работа/1_дата-аналитика
python3 sync_to_github.py
```

Или просто скажи AI-ассистенту: "Обнови папку для GitHub"
